API Overview

The Malawi LRL Data API provides programmatic access to the full catalog. Use it to search, filter, and retrieve metadata and download URLs for all collections. The API is read-only and requires an API key obtained through registration.

Base URL: https://api.malawi-lrl.org/v1

Authentication: Pass your API key via the X-API-Key header.

Endpoints

GET /collections List and filter all collections
ParameterTypeDescription
languagestringFilter by language code (e.g., chichewa, tumbuka)
media_typestringFilter by type: radio, news, books, surveys
providerstringFilter by provider slug
qstringFull-text search across collection names and descriptions
pageintegerPage number (default: 1)
per_pageintegerResults per page, max 100 (default: 20)

Example request:

# List all Chichewa radio collections curl -H "X-API-Key: YOUR_KEY" \ "https://api.malawi-lrl.org/v1/collections?language=chichewa&media_type=radio"

Example response:

{ "total": 12, "page": 1, "per_page": 20, "data": [ { "id": "col_radio_islam_01", "name": "Mawu a Anthu", "provider": "radio_islam", "language": "chichewa", "media_type": "radio", "format": "audio/wav", "hours": 245.5, "description": "Community call-in program..." } ] }
GET /collections/:id Get a single collection's full metadata

Returns complete metadata including file manifest, licensing, and download URLs.

curl -H "X-API-Key: YOUR_KEY" \ "https://api.malawi-lrl.org/v1/collections/col_radio_islam_01"
GET /providers List all data providers

Returns all organizations contributing data to the catalog.

curl -H "X-API-Key: YOUR_KEY" \ "https://api.malawi-lrl.org/v1/providers"
GET /languages List supported languages with statistics

Returns all languages in the catalog with collection counts and total hours/items.

curl -H "X-API-Key: YOUR_KEY" \ "https://api.malawi-lrl.org/v1/languages"
POST /collections/:id/download Request download URL for a collection
ParameterTypeDescription
formatstringDesired format: original, wav, mp3, json, csv
splitstringOptional: train, test, val (if pre-split available)
curl -X POST -H "X-API-Key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"format": "wav", "split": "train"}' \ "https://api.malawi-lrl.org/v1/collections/col_radio_islam_01/download"

Python SDK

Install the Python client for easier integration with ML pipelines:

pip install malawi-lrl from malawi_lrl import Client client = Client(api_key="YOUR_KEY") # Search collections results = client.collections.list(language="chichewa", media_type="radio") # Download a collection for training client.collections.download( "col_radio_islam_01", output_dir="./data", format="wav", split="train" ) # Stream for large datasets for chunk in client.collections.stream("col_mbc_news"): process(chunk)

About This Catalog

This catalog provides a centralized registry of low-resource language datasets from Malawi, intended for researchers and engineers working on natural language processing (NLP), automatic speech recognition (ASR), and large language model (LLM) fine-tuning for Malawian languages.

Malawian languages — including Chichewa (Nyanja), Tumbuka, Yao, Lomwe, and Sena — are significantly underrepresented in existing NLP corpora. This catalog aims to bridge that gap by indexing data from broadcasters, publishers, government institutions, and community organizations.

🎯 Purpose

Provide a single point of discovery for all available Malawian language data suitable for ML model training, so researchers don't have to hunt across disparate sources.

📋 Data Types

Audio (radio broadcasts, podcasts, recorded surveys), text (news articles, books, educational materials), and structured transcriptions paired with audio for ASR training.

🔑 Access

All metadata is freely browsable. Data downloads require registration and acceptance of the data use agreement. Some collections may have additional licensing restrictions from providers.

🤝 Contributing

We welcome new data contributions. If your organization produces Malawian language content and would like to make it available for research, please contact us to discuss ingestion and licensing.

📖 Citation

If you use data from this catalog in your research, please cite the catalog and the individual data providers. Citation formats are available on each collection's detail page.

⚖️ Ethics & Privacy

Household survey recordings have been reviewed for PII and anonymized. All data is shared under terms agreed upon with the originating institutions. Researchers must adhere to the data use agreement.

Apply to Access Datasets

Researchers, academic institutions, and organizations working on AI for social good can apply for access to these datasets. The application process involves submitting a research proposal, demonstrating ethical data handling practices, and committing to open-source publication of derived insights. We prioritize applications that show potential for significant impact on global development challenges and align with the Gates Foundation's mission.

Submit Application

Applications are reviewed on a rolling basis. Typical response time is 2–3 weeks.
